Player-Loader

Player Loader ReadME

1、初次加载，默认Flash 播放器： ① -> ②
（设置type为Flash）

2、Flash播放器切换至HTML5 播放器

  • 如果该浏览器支持HTML5播放器，则：③ -> ① -> ④
  • 如果该浏览器不支持HTML5播放器，则：③ -> ① -> ②

3、HTML5播放器切换至Flash播放器： ⑤ -> ① -> ②

4、再次加载时，会根据上次使用播放器类型：

  • 若为Flash播放器，则为： ① -> ②
  • 若为HTML5播放器, 则为： ① -> ④

5、外部调用播放器API时，则会根据播放器类型：

  • 若为Flash播放器，则为： ① -> ②
  • 若为HTML5播放器, 则为： ① -> ④
